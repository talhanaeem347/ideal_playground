# ideal_playground

    A Dating App for the Idealists and the Dreamers of the world. 

## Technologies Used
    
        Flutter
        Dart
        Firebase
        Google Cloud Platform
        Bloc Pattern

## Features

    Authentication
    User Profile
    User Matching
    User Chat
    User Search
    User Settings
    User Preferences
    Theme Switching


## Getting Started

    1. Clone the repository `git clone https://github.com/talhanaeem347/ideal_playground.git`
    2. Run `flutter pub get`
    3. Run `flutter run`
